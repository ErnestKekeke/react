# React Calculator App – Reviewer Instructions
# Overview

This project is a simple calculator built using React. It allows users to perform the four basic mathematical operations:
    Addition
    Subtraction
    Multiplication
    Division
It also includes buttons to reset the input field and reset the total result.


# Requirements
Before running the project, make sure the following software is installed:
Node.js
npm (comes with Node.js)

# Installation
Extract the zipped project folder.

Open a terminal in the project directory.

Run the following command to install dependencies:

npm install

This will create the node_modules folder and install all required packages.

# Running the Application
Start the React development server by running:

npm start

This will launch the application in your default browser.

If the browser does not open automatically, go to:

http://localhost:3000
