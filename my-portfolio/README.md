# My React Portfolio

This is a personal portfolio website built with **React** and **Chakra UI**.  
It showcases my projects, a short bio, and includes a contact form with validation.

## Features

- Landing section with avatar, greeting, and bio
- Projects section with cards in a grid
- Contact form with Formik and Yup validation
- Smooth scrolling between sections
- Global alert system for form submissions
- Responsive design

## Installation

1. Clone or download the project folder to your local machine.
...bash
git clone <your-repo-url>
cd my-portfolio

2. Install dependencies:

npm install
Make sure you have Node.js and npm installed on your machine.


## Running the App

Start the development server:
npm start

This will open the app in your browser at http://localhost:3000.

## Building for Production

To create a production build:
npm run build

The optimized build will be generated in the build/ folder.
